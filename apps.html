Coin iOS iPadOS

~/apps/coin.app
Privacy-focused spending tracker + AI backend · Swift / TypeScript

Spending tracking tuned for speed: a compact keypad sheet, voice capture, and receipt or screenshot OCR that drafts expenses through Coin AI, a managed backend on Cloudflare Workers gated by Sign in with Apple and App Attest hardware attestation. Analytics cover budgets, pace, calendars, trends and projections, with an Ask Insights chat that answers from on-device aggregates; your history never leaves the phone. Currently in TestFlight testing.

FEATURES
  • → Fast entry: keypad sheet, voice capture, receipt and screenshot OCR
  • → Monthly budgets with ideal-pace tracking and history
  • → Drag-to-read analytics charts, calendars, trends and projections
  • → Ask Insights chat over on-device aggregates via Coin AI
  • → Subscriptions, multi-currency, widgets and Siri Shortcuts
  • → Sign in with Apple + App Attest; keys live server-side only

Task iOS iPadOS macOS

~/apps/task.app
Privacy-focused task manager · SwiftUI / SwiftData

A task manager that treats the board as home: horizontally scrolling Kanban columns, cards that surface tags, dates, notes and reminders at a glance, and a Notion-style editor underneath: status, tags, working dates, due dates, and Markdown notes with tappable checklists. Local-first via SwiftData; iCloud sync is opt-in. A Mac-native edition (same SwiftData backend, rebuilt with Mac idioms) ships as Task for macOS. Currently in internal TestFlight testing; email me to become a tester.

FEATURES
  • → Multiple boards, each with its own groups, tags and defaults
  • → Kanban home: long-press to drag cards between columns
  • → Month calendar view with ranged tasks and a per-day +N pill
  • → Notion-style editor with live Markdown checklists
  • → Per-task local reminders with per-board reminder time
  • → Multi-board JSON export and non-destructive import

Body iOS iPadOS

~/apps/body.app
Privacy-focused health visualization · SwiftUI / HealthKit

A read-only window onto Apple Health: Activity Rings, a Readiness score built from personal baselines, sleep stages and vitals, workout history with calendars, and two-source comparison overlays for metrics like heart rate and steps. Widgets read a cached snapshot from the App Group; they never query HealthKit directly. Currently in TestFlight testing.

FEATURES
  • → Activity Rings card plus 14 metric cards with four-day previews
  • → Readiness score from personal baselines, with driver explanations
  • → Two-source comparison overlays on shared x-axis buckets
  • → Searchable workout history, calendar and type breakdown
  • → Sleep score, stage timeline and Apple-style Sleep Vitals
  • → Workout calendar and type-breakdown widgets
